#### 项目介绍
VSC 是一个基于vue的纯前端站群系统，支持多种模板和布局。响应式自适应移动和PC界面。<br/>
当前版本默认为【在线影视站的】模板布局： <br>
示例预览：http://v.meyho.cn/  <br>

- 风格主要偏手机版，PC版不作为重点考虑。<br/>
- 后续会加入各种界面模板机制以可以灵活的自定义个性化的设备版面适配。
- 可与任意语言开发的后台接口系统对接，快速组装前后端分离站群系统。
